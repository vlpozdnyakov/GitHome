# Инструкция по работе с Markdown 

## Выделение текста 
Чтобы выделить текст курсивом необходимо обрамить его звёздочкой (*). *Вот так*.

Чтобы выделить текст полужирным, необходимо обрамить его двойными звёздочками(**). Например **Вот так**.
